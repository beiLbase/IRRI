IRRI is a model that builds a prototypical network by querying the associations between instances and supporting instances. By integrating the relation information with the prototypical network through the adaptive module, the accuracy of relation classification can be improved.

The IRRI method was based on the paper “FewRel: A Large-Scale Supervised Few-Sample Relation Classification Dataset with State-of-the-Art Evaluation Results”.

The DOI of the orginal paper is:  https://doi.org/10.18653/v1/D18-1514

The link of the orginal code and the dataset: https://github.com/thunlp/FewRel
